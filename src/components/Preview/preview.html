Testing with Geno

  1. Navigate to project's local directory or server using the address bar
  2. Add the following lines to your HTML files:

    <link rel="stylesheet" type="text/css" href="./geno/geno.css" />

    <script src="geno/microsoft.cognitiveservices.speech.sdk.bundle-min.js"></script>

    <script src="geno/geno.js"></script>
  3. Build project using blue play button
