Testing with Geno

  1. Navigate to project's local directory or server using the address bar
  2. Add the following lines to your HTML file:

    <link rel="stylesheet" type="text/css" href="./geno/geno.css" />

    <script>
    import geno from "/path/to/geno/geno.js";
    geno.start(YOUR_DEV_ID);
    </script>
  3. Export functions mapped to voice commands:

    export { myVoiceFunction, myOtherFunction }
  4. Import geno in JavaScript files that call Geno functions:

    import { geno } from "/path/to/geno/geno.js";
  5. Build project using blue play button
