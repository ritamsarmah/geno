Getting Started

  1. Navigate to your project directory or server via the address bar.
  2. Add the following lines to your HTML file:

    <link rel="stylesheet" type="text/css" href="./geno/geno.css" />

    <script>
    import geno from "/PATH/TO/geno/geno.js";
    geno.start(YOUR_DEV_ID);
    </script>
  3. Build and integrate using the blue play button.


Creating Voice Commands

Function Commands

  1. Click the blue circle next to any function.
  2. Add sample user queries that could trigger the command. You must have at least 2, though adding more will improve accuracy.
  3. For any function parameters, click each sample query and map words to the correct parameters
  4. Click "Train Model" and wait for the model to finish training.
  5. Export functions mapped to voice commands:

    export { myVoiceFunction, myOtherFunction }
  6. Build and integrate using the blue play button.

UI Demo Commands

  1. Navigate to a webpage in your application.
  2. Click the mouse cursor button in the menu bar of the preview pane.
  3. Record a sequence of clicks on clickable webpage elements.
  4. Click the stop button in the menu bar of the preview pane.
  5. Add sample user queries and train the model.
  6. Build and integrate using the blue play button


Adding Voice Responses

  1. Import geno at the top of the file:

    import { geno } from "/path/to/geno/geno.js";
  2. Use geno.respond() to speak to the user.
  3. Use geno.ask() to ask the user a question and execute a callback on the response.
