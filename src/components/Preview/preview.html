Getting Started

Loading a project in Geno automatically generates a new directory called geno in the root of the project (don't modify any of the files in this directory). Before you can start creating voice commands, you'll need to integrate Geno into the project with the following steps.

  1. Add the following lines to an HTML file to include Geno:

    <link rel="stylesheet" href="./geno/geno.css" />

    <script type="module" src="/PATH/TO/geno/custom.js"></script>
  2. Build using the blue build button.
  3. Navigate to your application in a web browser.
  4. Start speaking to Geno by clicking the microphone button in the bottom right or Ctrl-`.


Creating Voice Commands

There are two types of voice commands you can create with Geno. Function commands trigger a function in a JavaScript file. Demo commands replay a sequence of UI interactions (clicks and typing) in the webpage.


Function Commands

  1. Click the blue circle next to any function.
  2. Add sample user queries that could trigger the command (minimum 2, though adding more will improve accuracy).
  3. For any function parameters, click each sample query and map words to the correct parameters
  4. Optionally configure multimodal input.
  5. Click "Train Model" and wait for the model to finish training.
  6. Export functions mapped to voice commands:

    export { myVoiceFunction, myOtherFunction }
  7. Build using the blue build button.
  8. Test your voice command within your webpage, by clicking the microphone button or the ` key.

Demo Commands

  1. Click the mouse cursor button in the menu bar of the preview pane.
  2. Record a sequence of clicks on clickable webpage elements. Typing into any text input element will generate a parameter that can be provided by the user to be automatically entered into the input element.
  3. Click the stop button in the menu bar of the preview pane.
  4. Add sample user queries and train the model.
  5. Build using the blue build button.


Configuring Multimodal Input

Multimodal input allows users to provide a command parameter by hovering over a single element, dragging over multiple elements, or highlighting text.

  1. Open the popover for a command and find the "Multimodal Input" section.
  2. Select a parameter from the dropdown to support multimodal input.
  3. Optionally specify the element type that should be recognized as context, by clicking the crosshairs and hovering over an element in the webpage to select it.
  4. Optionally specify the element attribute(s) to use for the parameter's value. After the previous step, click the element in the popover and select the desired attribute(s).
  5. Configure the rest of the command options as needed, and build.


Adding Voice Responses

You can directly add simple voice interaction functionality into your webpage.

  • Import geno at the top of the file:

    import { geno } from "/path/to/geno/geno.js";
  • Use geno.respond() to speak to the user.
  • Use geno.ask() to speak to the user, listen for a response, and execute a callback on the response.
