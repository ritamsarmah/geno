Getting Started

When you first load a project into Geno, it automatically generates a new directory called geno in the root of the project with all the files you need. Here is how to get started with configuring Geno.

  1. Add the following lines (after modifying the paths) to an HTML file to include Geno:
    <link href="/PATH/TO/geno/geno.css" rel="stylesheet" />

    <script type="module" src="/PATH/TO/geno/custom.js"></script>
  2. Create a voice command (see documentation for further help).
  3. Build using the blue build button.
  4. Navigate to your application in a web browser.
  5. Start speaking to Geno by clicking the microphone button in the bottom right or Ctrl-`.
